# Couleurs de texte

La couleur de texte doit traduire la hiérarchie : **principal**, **courant**, **complémentaire**, **état**. Évitez de choisir une couleur pour décorer un texte ; choisissez-la pour clarifier son rôle.

<h2 id="mise-en-avant">Mettre en avant un contenu</h2>

Utilisez `--color-text-primary` ou `--color-text-accent` pour attirer l’attention sur un mot, un statut de progression ou un lien important. Ne l’utilisez pas pour tout un paragraphe long.

```html
<p class="body-md">
  Le forfait inclut <strong style="color: var(--color-text-primary)">3 espaces actifs</strong>
  et une synchronisation quotidienne.
</p>
```

```html
<a class="body-md" style="color: var(--color-text-primary)" href="/settings">
  Configurer les préférences
</a>
```

<h2 id="contenu-moyen">Contenu moyen et complémentaire</h2>

Le contenu courant doit rester en `.body-md` ou `.body-sm`. Le contenu complémentaire (aide, metadata, note secondaire) doit être visuellement plus faible : `.caption`, `.label`, `.overline`, ou `--color-text-label`.

```html
<article>
  <h2 class="heading-lg">Rapport mensuel</h2>
  <p class="body-md u-mt-md">Résumé des indicateurs principaux.</p>
  <p class="caption u-mt-md">Mis à jour il y a 12 minutes.</p>
</article>
```

<h2 id="états">États et messages</h2>

Les couleurs d’état doivent rester fonctionnelles. `danger` signale une erreur ou un risque, `warning` une attention, `success` une validation et `info` une aide.

```html
<p class="body-sm-error">Impossible d’enregistrer : un champ requis est manquant.</p>
<p class="body-sm-warning u-mt-md">Attention : cette action modifiera les accès.</p>
```

<h2 id="exemples">Exemples copiables</h2>

```html
<div class="u-p-xl u-bg-neutral-100">
  <p class="label">Résumé</p>
  <h3 class="heading-md u-mt-md">Contenu prioritaire</h3>
  <p class="body-sm u-mt-md">
    Utilisez une couleur sobre pour la lecture et une couleur sémantique seulement pour signaler.
  </p>
  <p class="caption u-mt-md">Détail complémentaire ou contexte temporel.</p>
</div>
```

```html
<p class="body-md">
  Statut :
  <span style="color: var(--color-text-success)">opération terminée</span>
</p>
```
