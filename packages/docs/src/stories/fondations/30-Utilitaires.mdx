# Utilitaires (`u-*`)

Ces classes sont générées à partir de `packages/design-system/src/utilities/`. Le preview Storybook charge le **`dist/design-system.css`**, donc le même enchaînement **compilé** (tokens, reset, utilitaires, composants) que le package.

**Convention** : préfixe `u-`, sémantique **couleurs** (`u-bg-…`, `u-text-…`, `u-border-…`) ou **mise en page** (espacement, flex, position).
