# Design tokens

Les variables ci-dessous sont définies dans le design system (`fondation/token.scss`) et présentes dans l’**artefact** `dist/design-system.css` chargé par le preview. Les aperçus utilisent `var(--nom)` ou les classes `u-*` de ce bundle.
