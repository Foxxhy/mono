# Documentation des composants

Cette section documente les composants exposés par `dist/design-system.css`. Chaque composant possède une story de variations visuelles et une page MDX orientée HTML/CSS copiable, RGAA et patterns agnostiques.

## Sommaire

- [Organisation](#organisation)
- [Composants existants](#composants)
- [Patterns agnostiques](#patterns)
- [Règle RGAA commune](#rgaa)

<h2 id="organisation">Organisation</h2>

- Les stories montrent les variations et états visuels.
- Les MDX donnent le markup HTML minimal, les classes à copier et les obligations d’accessibilité.
- Les snippets restent agnostiques : HTML natif en priorité, JavaScript uniquement pour les comportements nécessaires.

<h2 id="rgaa">Règle RGAA commune</h2>

Un composant est utilisable seulement si son nom accessible, son rôle, son état et son comportement clavier sont corrects. Les classes CSS ne remplacent jamais les attributs HTML/ARIA nécessaires.
